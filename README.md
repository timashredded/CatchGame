Mobile 2D catch game prototype built with Unity.

## Description

This project is a simple mobile arcade game where the player controls a courier character and catches falling food orders while avoiding bad items.

The game was created as a gameplay and technical prototype focused on mobile controls, performance optimization and difficulty scaling.

## Features

- Touch-based horizontal movement
- Falling objects spawn system
- Dynamic difficulty scaling
- Lives system with heart UI
- Game Over screen with final score
- Optimized for mobile performance
- Android build available in Releases

## Tech Stack

- Unity 6000.3.4f1
- C#
- Android platform

## How To Run

1. Clone this repository
2. Open the project via Unity Hub
3. Switch build platform to Android
4. Build & Run

Or download APK from the Releases section.

## Project Purpose

This prototype was created to demonstrate:

- Mobile input handling
- Core gameplay loop implementation
- Performance optimization basics
- UI integration
- Difficulty progression logic
